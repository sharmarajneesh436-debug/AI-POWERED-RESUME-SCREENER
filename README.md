# 🤖 AI-Powered Resume Screener

An AI-powered Resume Screener that analyzes resumes, extracts candidate information, and matches them against job descriptions using Natural Language Processing (NLP). It helps recruiters quickly identify the most suitable candidates through automated resume scoring and ranking.

## 🚀 Features

- 📄 Resume upload (PDF)
- 🧠 AI-powered resume analysis
- 🎯 Job description matching
- 📊 Resume scoring and ranking
- 🏷️ Information extraction (Name, Email, Skills, Education, Experience)
- 🔍 Keyword matching
- ⚡ Interactive Streamlit interface

## 🛠️ Tech Stack

- Python
- Streamlit
- spaCy
- Pandas
- NumPy
- scikit-learn
- PyPDF2 / pdfplumber
- Regex
